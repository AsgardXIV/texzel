# Texzel

A texture encoding/decoding library for Zig.

## Supported Formats
| Format | Encode | Decode |
|---|---|---|
| Raw | ✅ | ✅ |
| BC1 (DXT1) | ✅ | ✅ |
| BC1a (DXT1) | ✅ | ✅ |
| BC2 (DXT3) | ❌ | ❌ |
| BC3 (DXT5) | ❌ | ❌ |
| BC4 | ❌ | ❌ |
| BC5 | ❌ | ❌ |
| BC6 | ❌ | ❌ |
| BC7 | ❌ | ❌ |

## Current State
Texzel is still very early in development and only supports a couple of formats currently.
The API is not stable, contributions are welcome.
